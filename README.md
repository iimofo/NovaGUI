# NovaGUI

Easy simple C++ GUI Lib

## Features
- window title
- button
- lable
- text input

## command to build
```
g++ myapp.cpp -std=c++17 -I"C:\glfw\include" -L"C:\glfw\lib-mingw-w64" -lglfw3 -lopengl32 -lgdi32 -static-libgcc -static-libstdc++ -o mygui.exe
```
! this is for the beginners in c++
